# sukidesu
勇気を出して想いを伝えるための、CLIツールです。

## インストール方法
以下のコマンドをターミナルで実行してください。

```bash
sudo curl -L https://raw.githubusercontent.com/shizu-na/sukidesu/main/sukidesu -o /usr/local/bin/sukidesu && sudo chmod +x /usr/local/bin/sukidesu
```

## 使い方

ターミナルでコマンドを打ち込んでください。

```bash
sukidesu
```

## オプション
- `-f`: 答えを急かします。
